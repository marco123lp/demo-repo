HOLA MUNDO DE GIT Y GITHUB

Comandos git

Para este`primer paso veamos los comandos de identificacion que debe utilizar
para autentificar al usuario vamos utilizar: git config --global user.name "marco123lp"
git config --global user.email "marcoadg70@hotmail.com"

Comando para ver con que usuario se encuentra autentificado

git config --global -l

Comando para hacer el primer init

Es decir para iniciar a un repositorio de git debemos iniciar el direcotrio (capete raiz) donde se aloja la pagina web en este caso
git init

Comando para ver si se han subido archivos al stage de git

git status

Comando para adicionar los archivos correspondientes de la pagina en desarrollo (index.html) en el stage

git add .
git add -A
git add index.html

Comando para dar luz verde o modificar de stage al repositorio de git

git commit -m "mensaje para saber que cambio se hizo y en que consiste el cambio"

Comando para ver los cambios que se hizo entre el proceso

git diff

Comando para revertir un paso

git checkout
nos permitira hacer un cotrol z a algun paso equivocado guardado

Comando para ver el historio de git

git log
este comando me permipe ver el historial de mis "commits" durante el trancurso de las modificaciones del codigo

Comando para ver el historio de forma mas ordenada

git log --oneline --decorate --all --graph

comando para remover un archivo del stage

git reset "nombre o ruta del archivo especifico"
este comando me pemite remover de forma independiente como grupal los archivos que quiero que no sean parte de este repositosrio.
Vale aclarar que solo elimina los archivos del stage y no asi del directorio original, solo lo aparta del stage del repositorio que posteriormente podemos volver agregarlo

Comando para colocar alias a ciertos comando especificos

git config --global alias.s "status"
git --global alias.co "commit
git --global alias.l "log"
est comando como su nombre indica me permite darla alias (apodos) a comandos que podrian ser bastante largos para asi poder abreviarlos
